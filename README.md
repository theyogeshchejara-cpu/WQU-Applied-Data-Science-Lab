# WQU-Applied-Data-Science-Lab
# WorldQuant University (WQU) - Applied Data Science Lab

Welcome to my repository containing the projects and assignments completed during the *Applied Data Science Lab (ADSL)* program at WorldQuant University. This repository showcases end-to-end data science workflows, from database querying to advanced statistical modeling.

## 🚀 Key Skills & Technologies
* *Languages:* Python (Pandas, NumPy, Matplotlib, Plotly Express)
* *Databases:* MongoDB (Aggregations, Queries, PyMongo)
* *Modeling:* Time Series Forecasting, Linear Regression, ARMA, AutoReg
* *Concepts:* Walk-forward validation, Timezone localization, Outlier removal, ACF/PACF plots

---

## 📂 Repository Structure & Projects

### 1. Data Wrangling with MongoDB (031-data-wrangling...)
* *Goal:* Connecting to MongoDB, exploring unstructured data formats, and preparing datasets for analytical pipelines.
* *Tools:* PyMongo, JSON parsing.

### 2. Linear Regression with Time Series (032-linear-regression...)
* *Goal:* Predicting asset prices and trends using classical machine learning regression techniques adapted for temporal data.

### 3. Time Series Forecasting Models (033-autoregressive... & 034-arma-models...)
* *Goal:* Building Statistical models to capture temporal dependencies and forecast values using lag parameters.
* *Models:* Autoregressive (AutoReg) & ARMA models.

### 4. Air Quality Forecasting in Dar es Salaam (035-assignment.ipynb)
* *Goal:* Predicting PM2.5 air pollution levels using walk-forward validation and optimizing lag hyperparameters (p) based on Mean Absolute Error (MAE).

---

## 📊 How to Run the Notebooks
1. Clone the repository: git clone https://github.com
2. Install dependencies: pip install pandas numpy statsmodels scikit-learn pymongo matplotlib plotly
3. Launch Jupyter Notebook or JupyterLab to execute the .ipynb files.
